Termos de Uso e Política de Privacidade

Este sistema de Gestão Financeira tem como objetivo auxiliar o usuário no controle de suas receitas, despesas e organização financeira pessoal.

1. Coleta de Dados

Coletamos dados fornecidos diretamente pelo usuário, como nome de usuário, e-mail e informações financeiras inseridas no sistema.

2. Finalidade do Uso

Os dados coletados são utilizados exclusivamente para funcionamento da aplicação.

3. Armazenamento e Segurança

Os dados são armazenados de forma segura, com medidas de proteção contra acessos não autorizados.

4. Compartilhamento de Dados

Não realizamos compartilhamento de dados com terceiros.

5. Direitos do Usuário
  • Acesso aos dados;
  • Correção;
  • Exclusão;
  • Revogação do consentimento.
6. Consentimento

Ao utilizar o sistema, o usuário concorda com os termos.
